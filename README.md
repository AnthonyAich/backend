# how to start the application


### the react application
#### install dependencies
```bash
yarn install
```

#### start the application
```bash
yarn start
```

#### test the application (cypress)
```bash
yarn cypress:open
```

---

### the express server
#### install dependencies
```bash
yarn install
```

#### start the server
```bash
yarn migrate
```
when asked if you want to create a new database, answer with
```bash
y
```

#### start the server
```bash
yarn start
```
